# LinkMasr - نظام إدارة المشاريع والعملاء

## نظرة عامة
LinkMasr هو نظام شامل لإدارة المشاريع والعملاء مصمم خصيصاً للشركات العربية. يوفر النظام مجموعة متكاملة من الأدوات لإدارة العملاء، المشاريع، المهام، والشؤون المالية.

## المميزات الرئيسية

### إدارة العملاء
- إضافة عملاء جدد (أشخاص، شركات، جهات حكومية)
- إدارة بيانات العملاء الشاملة
- تتبع حالة العملاء
- إدارة عناوين العملاء وعناوين الفواتير

### إدارة المشاريع
- إنشاء وإدارة المشاريع
- تتبع تقدم المشاريع
- ربط المشاريع بالعملاء
- إدارة مراحل المشروع

### إدارة المهام
- إنشاء وتوزيع المهام
- تتبع حالة المهام
- تحديد أولويات المهام
- ربط المهام بالمشاريع والمستخدمين

### الشؤون المالية
- إدارة الفواتير
- تتبع المدفوعات
- كشوف حسابات العملاء
- إدارة المصروفات

### إدارة المستخدمين
- نظام أدوار ومستويات صلاحيات
- إدارة ملفات المستخدمين الشخصية
- تتبع أنشطة المستخدمين

### مميزات إضافية
- التقويم وإدارة المواعيد
- نظام التنبيهات والإشعارات
- التقارير والإحصائيات
- نظام الدعم الفني
- واجهة باللغة العربية

## متطلبات النظام

- PHP >= 8.1
- Laravel 10.x
- MySQL >= 5.7
- Composer
- Node.js & NPM

## التثبيت

1. استنساخ المشروع:
```bash
git clone https://github.com/NourhanSami-25/LinkMasr.git
cd LinkMasr
```

2. تثبيت التبعيات:
```bash
composer install
npm install
```

3. إعداد ملف البيئة:
```bash
cp .env.example .env
```

4. تحديث إعدادات قاعدة البيانات في ملف `.env`

5. إنشاء مفتاح التطبيق:
```bash
php artisan key:generate
```

6. تشغيل الهجرات:
```bash
php artisan migrate
```

7. تشغيل البذور (اختياري):
```bash
php artisan db:seed
```

8. بناء الأصول:
```bash
npm run build
```

9. تشغيل الخادم:
```bash
php artisan serve
```

## الهيكل التقني

### المجلدات الرئيسية
- `app/Http/Controllers/` - المتحكمات
- `app/Models/` - النماذج
- `app/Services/` - خدمات الأعمال
- `resources/views/` - القوالب
- `database/migrations/` - هجرات قاعدة البيانات

### التقنيات المستخدمة
- **Backend**: Laravel 10, PHP 8.1+
- **Frontend**: Blade Templates, Bootstrap, jQuery
- **Database**: MySQL
- **Authentication**: Laravel Sanctum
- **Authorization**: Laravel Policies & Gates
